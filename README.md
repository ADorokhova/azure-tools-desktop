# Azure Tools Desktop

### Building the installer

#### OSX

```
npm run package:osx
```

Result: builds/Azure Tools-darwin-x64/azuretools.dmg.

#### Linux

```
npm run package:linux
```

Result: builds/azuretools-linux-ia32.
